# Restaurant Data Analysis Dashboard

A simple Flask-based dashboard for analyzing restaurant data.

## Features
- **Top Cuisines Analysis**: Identifies the most common cuisines in the dataset.
- **City Analysis**: Shows the city with the most restaurants and the highest average rating.
- **Price Range Distribution**: Visualizes the distribution of price ranges.
- **Online Delivery Status**: Breakdown of restaurants offering online delivery.

## Setup

1. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the Application**:
   ```bash
   python app.py
   ```
   The application will be available at `http://127.0.0.1:5000`.

## Configuration
- `.vscode/launch.json`: Contains VS Code debug configurations for both the Flask backend and Chrome frontend.
- `requirements.txt`: Lists all Python dependencies.
- `.gitignore`: Specifies which files and directories should be ignored by Git.

## Project Structure
- `app.py`: Main Flask application.
- `Dataset.csv`: Restaurant dataset.
- `templates/`: HTML templates.
- `static/`: CSS and other static assets.
